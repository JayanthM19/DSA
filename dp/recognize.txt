A simple rule to remember

Whenever converting memoization to tabulation:

If recursion calls solve(ind + 1, ...), fill the table backwards and use dp[i + 1].
If recursion calls solve(ind - 1, ...), fill the table forwards and use dp[i - 1].

DP Pattern Recognition

When you see:

Partition array into two groups
Minimize difference

Immediately think:

Subset Sum
Target = total//2
Find closest achievable sum
Answer = total - 2*sum

This is the same pattern as:

Partition Equal Subset Sum
Minimum Subset Sum Difference
Last Stone Weight II

all are subset-sum based problems.

The Golden Formula

Before writing any DP, answer these 3 questions:

1. What is my state?
2. What am I storing in that state?
3. If a state is impossible, what should it contribute?

Examples:

Problem	State stores	Impossible value
Coin Change	Min coins	inf
Target Sum	#ways	0
Jump Game I	Reachable?	False
Knapsack	Max profit	0
Ones and Zeroes	Max subset size	0
